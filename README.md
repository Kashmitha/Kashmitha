<details>
<summary>⚙️ <strong>One-time setup — GitHub Actions workflow for the snake</strong></summary>

<br/>

Create `.github/workflows/snake.yml` inside your `Kashmitha/Kashmitha` profile repo:

```yaml
name: Generate Snake Animation

on:
  schedule:
    - cron: "0 */6 * * *"
  workflow_dispatch:

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3

      - name: Generate snake SVGs
        uses: Platane/snk/svg-only@v3
        with:
          github_user_name: ${{ github.repository_owner }}
          outputs: |
            dist/github-contribution-grid-snake.svg
            dist/github-contribution-grid-snake-dark.svg?palette=github-dark

      - name: Push output to branch
        uses: crazy-max/ghaction-github-pages@v3.1.0
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

> Then: **Settings → Actions → General → Workflow permissions → Read and write → Save**
> Then run the workflow manually once from the **Actions** tab.
